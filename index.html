Repack Site Browser

Fast • Secure • Lightweight

Repack Site Browser is a modern Chromium-based Repacks site browser built with Delphi. It offers a clean interface, a customizable homepage, built-in downloads, and an allowed domains system for safer browsing to only trusted domains.

Why Repack Site Browser?

⚡

Fast Performance

Powered by Chromium for a smooth and responsive browsing experience.

🏠

Custom Homepage

Add, edit, remove and organize your favorite Repacks websites with ease.

⬇

Built-in Downloads

Download files directly using Chromium's internal download manager.

🛡️

Allowed Domains

Restrict browsing to approved websites for added security. Allow only trusted domains to ensure a safe browsing experience.
